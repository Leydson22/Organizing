# ✨ Overview

**ScanVid** is an intelligent local video learning platform built to organize educational content, courses, categories, and video lessons in a modern and responsive interface.

The application combines:

- 🎬 **Cinema-style video player**
- 📚 **Smart course organization**
- 🌙 **Dark & Light premium themes**
- 🧠 **Intelligent sidebar navigation**
- ⚡ **Glassmorphism UI**
- 📱 **Responsive design**
- 🗂️ **Dynamic folder organization**
- 🎯 **Learning-focused experience**
